# Big Data Lab Exercises - HCMUT

Welcome to the Big Data Lab Exercises repository! This repository contains a series of laboratory assignments completed as part of the Big Data coursework at Ho Chi Minh City University of Technology (HCMUT).

## Overview

This repository is structured into separate folders, each representing a specific lab exercise focusing on various big data technologies and concepts, such as Apache Spark, Apache Kafka, and data streaming.

### Repository Structure

*   **[LAB01](./LAB01/)**: Introductory exercises on Big Data processing. Please navigate to this folder for specific materials and Jupyter notebooks.
*   **[LAB02](./LAB02/)**: Spark Structured Streaming Analysis involving real-time analytics, data ingestion from Kafka, and complex transformations using Apache Spark.

## Detailed Instructions

To get started with any of the lab exercises:

1.  Navigate to the specific lab folder (e.g., `LAB01`, `LAB02`).
2.  Inside each folder, you will find a dedicated `README.md` file (or instructional notebooks) that provides detailed guidelines, the topic's content, and specific instructions on how to run the code.
3.  Follow the prerequisites, setup guide, and execution steps outlined in the respective lab's documentation.

## Prerequisites

General requirements to run the exercises in this repository (specific versions and configurations may be detailed in each lab folder):

*   Python (3.12+)
*   Apache Spark (4.0.1+)
*   Apache Kafka (3.6.0+)
*   Jupyter Notebook
*   Docker & Docker Compose (optional, for environment setup as provided in `docker-compose.yml`)
